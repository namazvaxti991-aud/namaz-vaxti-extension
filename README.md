# Namaz Vaxti Azerbaijan – Prayer Time Extension

This is a browser extension that provides accurate daily prayer times (Namaz vaxti) for different cities in Azerbaijan.

🌐 Website: https://namazvaxtiaz.com

---

## 📌 About the Project

This extension is designed for Azerbaijani users to easily check prayer times in their local language without visiting websites repeatedly.

It currently supports multiple cities across Azerbaijan and aims to provide a fast and simple user experience.

---

## 🚀 Features

- Accurate daily prayer times
- Multiple cities in Azerbaijan
- Azerbaijani language support
- Simple and clean interface
- Fast access without opening websites

---

## 🌍 Cities Covered

- Baku  
- Ganja  
- Sumqayit  
- Shaki  
- Yevlakh  
- Shirvan  
- Mingachevir  
- Quba  
- Khachmaz  

---

## 🔗 Live Website

You can check prayer times online here:  
👉 https://namazvaxtiaz.com

---

## 🧩 Extension Purpose

The goal of this extension is to make prayer times easily accessible for users in Azerbaijan, especially for those who prefer using Azerbaijani language tools.
