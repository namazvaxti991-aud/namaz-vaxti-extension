Namaz Vaxti Azerbaijan – Prayer Time Extension

This browser extension provides accurate daily namaz vaxti for different cities in Azerbaijan. It helps users easily check prayer times without visiting websites again and again.

About the Extension

This extension is specially built for Azerbaijani users who want quick access to prayer times in their local language. It is fast, simple, and easy to use.

Features

  • Accurate daily prayer times
  • Supports multiple Azerbaijan cities
  • Simple and clean interface
  • Fast access without opening websites

Cities Covered

  • Baku
  • Ganja
  • Sumqayit
  • Shaki
  • Yevlakh
  • Shirvan
  • Mingachevir
  • Quba
  • Khachmaz
